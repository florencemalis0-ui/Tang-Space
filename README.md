# tangruiyang's Homepage

个人主页，基于 React + Vite + TypeScript 构建，部署在 GitHub Pages。

## 技术栈

- React 18 + TypeScript
- Vite
- 原 vno.css 样式体系

## 特性

- Bing 每日壁纸背景（GitHub Actions 自动更新）
- 一言（Hitokoto）随机句子
- 头像 3D 悬停动效
- 移动端自适应

## 本地开发

```bash
npm install
npm run dev
```

## 构建部署

```bash
npm run build
```

构建产物在 `dist/`，部署到 GitHub Pages。
